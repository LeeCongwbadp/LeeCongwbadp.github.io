人物 介绍

李白

李白（701年2月28日—762年12月），字太白，号青莲居士，祖籍陇西成纪，出生于蜀郡绵州昌隆县（一说出生于西域碎叶）。唐朝伟大的浪漫主义诗人，他的诗作被广泛传诵和研究，被视为中国文学珍宝之一。

屈原

屈原（约公元前340年-278年），中国古代著名的诗人、政治家和文化名人。楚国人，被誉为中国楚辞的创始人和代表人物。楚辞是中国文学史上的重要文学流派之一，以其独特的抒情风格和深邃的哲理而闻名。

鲁迅

鲁迅（1881年-1936年），原名周樟寿，后改名周树人，字豫山，号鲁迅，是中国现代文学和思想史上的重要人物，也是中国现代文化发展的杰出代表之一，是中国现代文学史上最有影响力的作家和文化评论家之一。

王小波

王小波（1952年-1997年），原名王曙光，是中国当代著名作家。他写作风格独特和对社会现象观察犀利，被誉为中国当代文学界的"反文艺青年"，作品多样且具有创新性，涵盖小说、散文、随笔、杂文等多个文学领域。

朱自清

朱自清（1898年-1948年），字梅花，号自清，是中国现代著名的作家、教育家和翻译家，中国现代散文的代表人物之一。其作品以纯美、清新、婉约的风格而闻名，对中国现代散文的发展产生了深远的影响。

闻一多

闻一多（1899年-1947年），字义夫，号雪莲，是中国现代文学史上重要的学者、思想家和教育家之一。中国现代散文运动的先驱者之一，新月派文学团体的核心人物。作品具有深厚的思想内涵和鲜明的人文关怀。

胡适

胡适（1891年-1962年），原名适之，字适之，号适庵，是中国近代著名的文学家、思想家和教育家，也是新文化运动的重要领导人之一。早年留学美国，深受西方文化和思想的影响，主张借鉴和吸收西方科学。

王国维

王国维（1872年-1927年），字静安，浙江杭州人，是中国现代文化史上的重要人物之一。他是中国近代文化界的学者、教育家、思想家和诗人，也是“新文化运动”时期的重要推动者之一。

一、主题描述

该网站采用HTML、CSS、JavaScript“网页三剑客”实现了以上静态网页，由于本人对文学和相关人物的事迹十分感兴趣，因此我的目标是通过这个网站，让其他人也能了解中国历史名人， 探究中国文学的精华，促进文学知识的传播和交流。无论是学术研究者、学生还是文学爱好者，都可在该网站上找寻到属于诗人的那份浪漫、属于作家的那种深邃、属于民国人的那段历史。

诗人—— 在那个黄昏的时刻，诗人如夕阳一般灿烂。他的眼睛闪烁着星辰的光芒，被智慧和梦想所温暖。 他的笔触如花瓣般轻盈，舞动在纸上，绘出诗的画卷。每一个字，都流淌着诗意的血脉，每一行，都跳动着音符的旋律。 他捕捉思绪的飞鸟，将它们安放在诗篇的枝头。他把花香和月光融入文字的色彩，用文字的魔力传递心灵的诉求。 诗人如晨曦中的水滴，含着大地的清新和温润。他的诗语如清泉般流淌，滋润人们内心的干渴与沉寂。 他以诗为羽翼，飞越时光的边界，寻找着美和真理的永恒存在。他的灵感是天上的星星，闪耀着创造和感知的光辉。 诗人，那位创造文字之美的艺术家，用诗歌雕刻情感的纹理，他是语言之王，灵感之巨星，为世界的荒芜带来了盛放的花期。

作家—— 在寂静的黑夜里燃起了星火，用文字为世界织起了华美的画卷。他们是思想的航海家，探索真理的探险者，在纸张上驰骋着自由的思绪与想象。 他们的思维如闪电般犀利，点燃了读者内心的灵感之火。他们用笔触凝聚着人类的智慧和情感，将文字编织成富有意义的故事和抒情。 每一个字都是他们的心灵乐章，每一段话都流淌着感悟的泉水。他们的笔触如尖锐的刀锋，剖开现实的表象，揭示内心的秘密。 作家是时光的见证者，记录者，用文字把过去的岁月定格为永恒。他们 心灵的解码者，启迪者，用文字为人们揭示生命的奥秘与真相。 他们善于观察，捕捉细微的细节，在平凡的日常中寻找不平凡的灵感。他们执着于文字的雕琢与调和，用文笔绘画出世界的多彩与多姿。 作家，那位用灵魂书写的艺术家，用文字开启了想象的大门。他们用才华和汗水炼制了每一行文字，在纸张上留下永恒的足迹和意义。 作家们，以他们的笔为羽翼飞翔，跨越了时空的界限，触及人心的温暖。他们是信念的倡导者和梦想的守护者，用文字点燃了世界的热情和希望。

民国人—— 那时的人物，在民国的岁月里闪耀着光芒，他们是那个时代的见证者和缔造者。他们的衣着典雅，举止风度翩翩，在变幻的时代中展示着坚韧的品格。 他们执着于理想与追求，用无尽的努力创造着辉煌的事业。他们是知识的守望者和智慧的传播者，用自己的才华开启了人们的思想之门。 那时的女性，温柔而坚毅，她们的声音如柔风般轻盈婉转。她们敢于追求平等和自由，在尘封的传统中展现出勇敢的姿态。 那时的男性，绅士而豪放，他们的目光中蕴含着自由的热忱。他们奋发向前，为国家和民众奉献，不畏艰险，勇于担当起历史的责任。 民国时期的人物，彰显着民族的骄傲与自豪，他们的心怀着对国家和人民的无限热爱。他们是智慧的火炬，在黑暗中闪耀，引导着国家走向文明和繁荣的道路。 乘风破浪，他们勇敢地奏响时代的交响，他们的思想与行动改变了整个社会。他们是时代的先驱，人类发展进步的推动者，永远留在历史的篇章中，永不磨灭的记忆。

二、开发过程

1.版块划分

在确定主题后，本着向其他人介绍自己爱好的意愿，因此选择介绍一些自己比较喜欢的人物，以及描写这些人物的文章，最后设置了“人物介绍”和“美文欣赏”页面， 其次本着希望结交更多志趣相投的朋友的原则，设置了一个“青春会客厅”页面，愿有更多人能够喜欢上文学，喜欢上这些人物，以及他们极具魅力的人生故事。此外， 本网站中还设置了“主页”、“个人总结”等页面，使整个网站更系统全面，同时也方便老师直接查看“课程结业报告”。

说明：整个网站由6个页面设计而成，每一个页面由一个section标签设计而成。section是HTML中用来创建一个独立的区块或内容部分的标签。它可以用于组织和 结构化页面中的不同部分，例如文章、章节、侧边栏、页眉、页脚等。此处用section来构建一个页面，并用CSS样式使页面能够“全屏”展示。

2.“主页”页面的设计

2.1 设计过程

该界面在HTML中由一个“section”标签（该标签设置class为home），两个“div”标签（class分别为content和image）、一个“h3”标签、一个“p”标签、八个“br”标签、一个”a“标签和一个”img“标签组成。 在CSS样式设计中，首先设计了整个section的背景页面，使用“class选择器”，即“.home{}”,其中使用“弹性盒子”flex展示，设置flex-wrap属性设置弹性容器内的元素在空间不足时进行换行显示，设置background 为页面增设背景图片等。此外，在section下的div标签内，仍然采用“class选择器”，其中设置了“flex: 1 1 40rem”，表示一个用于弹性盒子布局的缩写属性。它表示弹性元素将根据可用空间进行拉伸，但不会缩小， 初始大小为40rem，还设置了“padding-top: 6.5rem”，表示设置顶部内边距为6.5rem，为内容提供间距。随后还设置了图片的样式，使用圆形展示，且使用”animation-name: float“， 指定要应用的动画名称为 “float”，它将匹配在CSS中定义的带有 @keyframes 规则的动画效果，从而实现图片上下浮动的效果。不仅如此，该页面中还设置了字体的样式，首先需要引入从fonts官网中引入 CSS样式，随后在CSS页面中设置相应标签中的字体样式属性“font-family”，引入需要的字体样式即可。除此之外，本网站的按钮设计遵循一个原则，即遵循设置的相应的内边距padding，边框样式border， 光标形状变化cursor以及元素的定位隐藏等，当光标聚焦时，此时按钮背景颜色将变红，且通过transition属性定义过渡动画效果。其次，按下该界面的按钮可直接跳转到“人物介绍”界面，其实现原理是在HTML中 内部实现了一个超链接标签，其href属性设置为“#speciality”，其中，speciality为section标签的class属性值。

2.2 遇到的问题及解决方法

该阶段主要在字体样式的设计上花费了较多心思，最开始字体没有设置成功，后来发现设置样式的位置有错误，将位置改成对应字的标签位置处即可完成字体的改变。再者就是在图片的浮动设计上算是比较具有创新性的点。

3.“内容概览”页面的设计

3.1 设计过程

该界面整体仍然使用“section”标签（该标签设置class为step），仍采用flex布局，设置了该界面的字体样式，设置了图片的圆形样式，有所不同的是，在设置图片圆形样式的同时，为该图片的外围增加了白色的 边框。其实现原理是利用”border: 1rem solid #fff“设置图片的边框样式为宽度为1rem的实线边框，颜色为白色。此外，还通过“box-shadow: 0 .5rem 1rem rgba(0,0,0,.1)”为图片添加一个盒子阴影效果， 偏移量为垂直方向0，水平方向0，模糊半径为0.5rem，阴影颜色为 rgba(0,0,0,.1)。

3.2 遇到的问题及解决方法

该阶段遇到的问题是没有将图片完全设置为圆形，可能原因是图片长宽比例差距太大，导致不能将图片设置为圆形，但却又重新设置图片的宽度和宽度为100%，但仍不能显示圆形，正确原因暂时不太清楚，仍在探索中。 此外，该界面由于内容太少，无法使该section占满整个屏幕，在设计过程中我在这个问题花费了较多的时间，首先想到在CSS中设置元素高度， 即在CSS中使用单位“vh”，用于设置“视口高度”。以下为设置元素高度占满屏幕的代码： “.element{height: 100vh; /* 设置元素高度为100视口高度 */}”。如此，“.element”元素的高度将会等于当前屏幕的高度，从而占满整个屏幕。 但这并没有完美地解决我的问题，如移动设备上的地址栏伸缩会导致元素高度不准确。因此最终决定使用JavaScript来动态计算页面高度，以确保元素占满整个屏幕。 以下为使用JavaScript动态计算元素高度的代码： ”window.addEventListener('resize', function() { var windowHeight = window.innerHeight; var element = document.querySelector('.element'); element.style.height = windowHeight + 'px'; });var windowHeight = window.innerHeight; var element = document.querySelector('.element'); element.style.height = windowHeight + 'px';“。 上述代码使用`window.innerHeight`获取窗口的视口高度，并在窗口大小改变时更新元素的高度。同时，还在页面加载时初始化了一次高度。这样，无论窗口大小如何变化，`.element`元素的高度都会保持与当前屏幕高度一致。 不仅如此，在设计该页面时可谓“命运多舛”啊！在导入图片时，明明已经设置好了flex弹性盒子，但四张图片却无法均匀地显示在一行上， 在使用”display: flex“、“justify-content: space-between”的基础上，最后通过调整图片之间的间距，即通过设置“flex: 1” 保证图片在容器内平均分布，并通过“margin”属性调整图片之间的间距，使用“max-width: 100%”和“height: auto”用于确保图片在容器内按比例缩放，并且不会变形。

4.“人物介绍”页面的设计

4.1 设计过程

该界面整体仍然使用“section”标签（该标签设置class为speciality），仍采用flex布局，设置了该界面的字体样式，设置了图片的相关样式，有所不同的是， 在界面中还引入了图片与文字的交替显示。其实现原理是设计一个CSS选择器，当有鼠标悬停在具有“.box”类的“.box-container”元素的内部时，对“.content”元素应用一些样式。 具体来说，当鼠标悬停在“.box-container”元素内部的“.box”元素上时，.content元素将会应用“transform: translateY(100%)”样式。这个样式规则将使“.content” 元素沿垂直方向向下平移100%，也就是从其正常位置向下移动到一个等于自身高度的位置，以此便实现了图片与文字的交替显示。此外，该界面使用了“Font Awesome”中的人物图标， 其实现过程是在HTML中引入Font Awesome图标库的脚本引入代码，随后在HTML标签中使用Font Awesome相应图标的样式类，如”fas fa-user-secret“即可显示图标。

4.2 遇到的问题及解决方法

在该界面的设计过程中，使用Font Awesome无法正常导入图标，经过自己不懈的探索，最终发现了解决办法。首先需要在“fontawesome.com”官网中注册登录后获得kit，随后将此 kit引入网页的HTML中，该脚本链接提供了一个更方便的方式来使用Font Awesome图标，通常用于使用Font Awesome Kits进行图标自定义的情况，其大致内容如下 “|script| src="https://kit.fontawesome.com/c5b69774ce.js" crossorigin="anonymous"|/script|”（“<”已替换为“|”）。通过搜集资料发现， Font Awesome Kits 提供了更高级的功能，例如使用特定的图标集、自定义CSS、变量等功能。

5.“美文欣赏”页面的设计

5.1 设计过程

该界面整体仍然使用“section”标签（该标签设置class为gallery），仍采用flex布局，设置了该界面的字体样式，设置了图片的相关样式，其设计形式与“人物介绍”页面一致， 但有所不同的是，“人物介绍”页面默认显示文字，当鼠标置于其上时，才会显示人物图片；而“美文欣赏”页面默认显示为图片，当鼠标置于其上时，才会显示相关文字、超链接式按钮等。 其实现方式是在CSS中设置class选择器，当鼠标悬停在具有“.box”类的“.box-container”元素的内部时，对“.image”元素应用一些样式，即当鼠标悬停在“.box-container” 元素内部的“.box”元素上时，“.image”元素将会应用”top: 0“样式，该样式规则将使“.image”元素的垂直位置调整为距离父元素顶部为0的位置，也就是将其位置移动到上方，以此 便实现了“人物介绍”页面中默认展示文字的的效果；而在“美文欣赏”页面的CSS样式中，则是将“.content”元素设置为“top: 0”的样式，因此会默认展示图片。

5.2 遇到的问题及解决方法

由于该阶段主要遵循着以上界面的设计原则，因此在设计该界面时并没有遇到太多的困难和问题。

6.“青春会客厅”页面的设计

6.1 设计过程

该界面整体仍然使用“section”标签（该标签设置class为order），仍采用flex布局，设置了该界面的字体样式，设置了图片的相关样式。该界面主要实现了一个表单的提交， 该表单的输入标签input包括text、number、password等类型的文本。在用户填写好信息（或者不填也行，功能没有设置完整）后，可选择“立刻加入”按钮，此时窗口会弹出 一个提示框，提示“欢迎你！你已成为大家族中的一员！”，并回退到“内容”概览页面，这是由于在HTML和JS中设计了相关的鼠标点击事件。

6.2 遇到的问题及解决方法

在本阶段中，本想着有password类型的输入框，是否可以设置一个“密码密文和明文的切换”功能，并在该输入框内设置一”眼睛“图标。理想是丰满的，但现实是骨感的。 实施过程中，虽可实现密文和明文的切换，但无法解决当输入内容为空且此时以明文形式呈现时，输入框此时本该以密文的形式接收数据的问题。其次，最初的眼睛图标 显示于输入框外，样式很丑陋，便想着如何将眼睛图标内嵌至input中，可是经过很多次尝试，最后都以失败告终。

7.“其余细节的设计

7.1 悬浮框的设计

“悬浮框”，即当页面未处于首页时出现在页面右侧的悬浮按钮，当点击它时，页面将跳转至首页，且该按钮将立即消失。该按钮的实现原理与首页“时光穿梭……”按钮实现的原理一致， 不过此处的按钮有了而更多的细节，除了在其CSS页面中设置其固定样式外，还需要保证当前进入首页时，该按钮会立即消失，其实现原理是当元素同时具有“#scroll-top ID”和“.active类”时， 将应用“top: calc(100% - 12rem)”这一样式。该规则使用了CSS calc()函数，计算出该元素相对于其父元素顶部的位置。 在这个特定的规则中，元素的位置将被计算为其父元素高度的百分之100减去12rem。这意味着元素将被定位在距离其父元素底部12rem的位置。 如此便可实现一个滚动到顶部的按钮，在滚动到第二个页面时显示，并定位在页面底部的固定位置。

7.2 菜单栏的设计

菜单栏的设计主要说明一下其CSS样式的设计，该菜单栏为响应式菜单栏，在CSS中建立了一个媒体查询，当屏幕宽度小于等于768像素时应用一些样式。 媒体查询允许在不同的屏幕尺寸上应用不同的样式。 具体来说，当屏幕宽度小于等于768像素时，以下样式规则将生效： ①#menu-bar 元素将显示（display: initial;）。 ②header .navbar 元素将设置为绝对定位，并位于父元素顶部100%的位置，全宽度（left: 0; right: 0;）。其背景将设置为#f7f7f7，顶部将有一个0.1像素粗的黑色边框，并且clip-path属性将定义为一个四边形，覆盖了元素本身的整个区域。 ③header .navbar.active 元素将具有不同的 clip-path 规则，更改形状以显示不同的样式。 ④header .navbar a 元素将有一些内边距和边框样式。 这些样式规则用于创建响应式的菜单栏，使其在小屏幕上以不同的方式呈现，例如显示下拉菜单或调整大小和样式。

7.3 加载页面的设计

该页面的实现原理是在JS中定义一个JavaScript函数 loader()，它通过添加一个CSS类来执行一个动画效果，从而隐藏一个加载器。 具体来说，这个函数使用document.querySelector('.loader-container')来获取页面中具有.loader-container类的元素。 然后，它使用classList.add('fade-out')将fade-out类添加到该元素的classList中。 通过添加fade-out类，该元素将应用一个CSS动画效果，使其逐渐消失或淡出。这个动画效果通过在CSS中定义fade-out类的相关样式规则来实现。 这个函数在页面内容加载完成后调用，以隐藏加载器并展示页面内容。
